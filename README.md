# LibrarIQ — AI-Powered Library Management System

A full-stack library management system built with React (frontend) and Flask (backend).

## Features
- 📚 Book catalog with search & filter
- 📖 E-Book reader (for unavailable physical copies)
- 🔐 Role-based auth: Student | Staff/Teacher | Librarian
- 📷 Barcode scanner for book issue
- ✍️ Manual Entry module (when scanner fails)
- 🤖 AI chatbot assistant
- 📋 Book procurement request system (Staff)
- 🌙 Aesthetic library background

```bash
# Frontend
cd frontend
npm install

```
